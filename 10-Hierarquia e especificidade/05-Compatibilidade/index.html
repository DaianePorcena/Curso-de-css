Prefixos no Css

Mecanismos originalmente usados para permitir que alguns navegadores implementem sua própria versão de um recurso Css enquanto essa tecnologia ainda está em fase de testes sem esntrar em conflito com a implementação de outros navagadores.

Exemplos de prefixos:

  • -moz-: Firefox
  • -webkit-: Crhome/Edege/Opera/Saari
  • -ms-: IE (versões antigas)
  • -o-: Opera (versões antigas)
